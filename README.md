# student-management-system

This is a simple Python project made by me that manages student records using a CSV file.  
It allows adding, viewing, searching, updating, and deleting students.

## Features of the project
 can add Student name
 can view All Students  
 Search the student with the help of roll Number  
 Update Student  
 Delete Student  
 Stores data in student.csv file

## How to Run the code
1. Create 3 files in the same folder:  
   student.py  
   students.csv (keep it empty)  
   README.md
2. Open terminal in the folder.
3. Run
