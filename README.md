# Student Management System made using python

This is a simple Python project made by me that manages student records using a CSV file.  
It allows adding, viewing, searching, updating, and deleting student details.

## Features of this project
 Add Student  
 View All Students  
 Search by Roll Number  
 Update Student  
 Delete Student  
 Stores data in students.csv file

1. Create 3 files in the same folder:

    README.md
