# Statement of project Student management system
## 1 Problem statement
     In schools and college managing a lot of students is very difficult and it is very time 
     consuming so this student management system is very useful for that.
## 2 Scope of the project
     the project works on command line (terminal)
     everything will be saved in csv file
     provides basic operations like add student,add roll no, view all students and many other things
     this project is made only with the help of python
     made for beginner level record management
## 3 Target users
     can be used by students teachers and professors
## 4  High level features
     Add student
     view all students
     search student
     update student
     delete student
     csv based storage file
